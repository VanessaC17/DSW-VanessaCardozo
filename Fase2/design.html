Etiquetas Básicas de HTML, atributos y valores

El proposito de este sitio es informativo y de aprendizaje sobre las etiquetas que existen en HTML atributos y valores que tiene cada una de las etiquetas, los documentos html (HiperText Markup Language o Lenguaje de Marcas de Hipertexto ) como su nombre lo dice son archivos de texto, cada pagina tiene una serie de elementos que la compone, cada etiqueta tiene mucha y diferente informacion cada una le indica al navegador a la pagina su principio y fin.

Etiquetas Básicas de HTML Atributos Valores
<head>    
<!DOCTYPE html>    
<span>    
<br>    
<footer>    
<div> id caja
<a> href https://radios.com.co/
<strong>  title fuerte
<H1> hasta <H6> align left, center, right, justify
<img>  src= imagen.png
<ol> compact, reversed, start. compact, reversed, number.
<p> class= titulo
<body>  alink, background, bgcolor, link, text, vlink URL, color.
<script> async, charset, defer, src. URL
<table> align left, center, right, justify
Algunas de las etiquetas y su especificacion:
* < body> para el contenido
* < head> para información sobre el documento
* < div> división dentro del contenido
* < a> para enlaces
* < strong> para poner el texto en negrita
* < br> para saltos de línea
* < H1>… < H6> para títulos dentro del contenido
* < img> para añadir imágenes al documento
* < ol> para listas ordenadas, < ul> para listas desordenadas, < li> para elementos dentro de la lista
* < p> para parágrafos
* < span> para estilos de una parte del texto
